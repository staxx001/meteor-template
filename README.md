# Meteor Template
A Meteor Project Template using Blaze Templates and Flow Router.

## Installation

In your projects folder run:
```
git clone https://github.com/incode-dev/meteor-template.git
```

Change directory to the project:
```
cd meteor-template
```

Install npm modules:
```
meteor npm install
```

Start Meteor:
```
meteor
```
